# is-valid-github-url

> Check if passed link is a valid GitHub url (belongs to github.com and doesn't return an error)

## Install

```
$ npm install --save is-valid-github-url
```

## Usage
```js
var isValidUrl = require('is-valid-github-url');
```

## API

### `isValidGithubUrl()`

Check if passed link is a valid GitHub url (belongs to github.com and doesn't return an error)
