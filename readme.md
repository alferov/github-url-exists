# github-url-exists

> Check if passed GitHub link exists and doesn't return an error

## Install

```
$ npm install --save github-url-exists
```

## Usage
```js
var isValidUrl = require('github-url-exists');
```

## API

### `isValidGithubUrl()`

Check if passed GitHub link exists and doesn't return an error
