# is-valid-github-url

> Check if a passed string is a valid GitHub URL (doesn't return an error)

## Install

```
$ npm install --save is-valid-github-url
```

## Usage
```js
var isValidUrl = require('is-valid-github-url');
```

## API

### `isValidGithubUrl()`

Check if a passed string is a valid GitHub URL (doesn't return an error)
